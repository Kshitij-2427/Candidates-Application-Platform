Follow these steps to run the project locally:(Make sure to use Node Version 18)

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Kshitij-2427/Candidates-Application-Platform.git
   cd your-repo

2. **Install Dependencies:**
npm install

3. **Run:**
ng serve

3. **Check at:**
http://localhost:4200/
